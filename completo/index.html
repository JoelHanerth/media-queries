Layout responsivo com media queries

Este projeto usa a abordagem mobile-first: estilos básicos funcionam bem em celulares e, usando media queries, o layout se adapta para telas maiores (tablet/desktops).

Card 1

Um exemplo de card. Em telas pequenas os cards ficam empilhados; em telas maiores aparecem lado a lado.

Card 2

Você pode ajustar a propriedade grid-template-columns dentro da media query para controlar quantas colunas aparecem.

Card 3

Tente redimensionar a janela do navegador para ver as mudanças — ou abra no celular para a versão mobile.
